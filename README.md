# Ecommerce Order Management System (Java)

## Project Description
This is a console-based ecommerce order management system developed in Java. 
The system handles product management, cart operations, order placement, 
payment simulation, inventory reservation, concurrency handling, and low stock alerts.

## Features
- Add and View Products
- Add to Cart / Remove from Cart
- View Cart
- Place Order with Coupon Discounts
- Cancel Order
- View Orders
- Search Order
- Low Stock Alert
- Concurrent Purchase Simulation
- Inventory Reservation System
- Payment Simulation
- Concurrency Handling using Locks

## Technologies Used
- Java
- OOP Concepts
- Collections Framework
- Concurrency (ReentrantLock)
- Git
- GitHub

## How to Run
1. Open project in Eclipse
2. Run EcommerceEngine.java
3. Use menu options in console
